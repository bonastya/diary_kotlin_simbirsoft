# diary_kotlin_simbirsoft
### Приложение-ежедневник с отображением и добавлением дел. 
Приложение содержит два экрана  
-для выбора даты и отображения дел в виде блоков  
-для добавления дела  

Для локального хранения используется бд Realm.
